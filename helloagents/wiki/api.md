# API 手册

## 概述
本服务对外暴露的 HTTP 接口以“企业微信回调入口”为主，供企业微信服务器调用；同时提供健康检查接口便于部署与监控。

## 认证方式
- **企业微信回调验签/解密:** 基于 `msg_signature` + `timestamp` + `nonce` 校验签名，并使用 `EncodingAESKey` 解密消息体。
  - 回调 POST 解密成功后默认返回 `success`，业务回复通过“发送应用消息 API”异步下发。

---

## 接口列表

### 基础

#### [GET] /healthz
**描述:** 健康检查

#### [GET] /readyz
**描述:** 就绪检查（依赖项就绪后返回 200）

### 企业微信

#### [GET] /wecom/callback
**描述:** 回调地址验证（企业微信“设置接收消息服务器”时的校验请求）

#### [POST] /wecom/callback
**描述:** 接收企业微信消息/事件回调（XML 加密载荷）
